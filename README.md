# Station widget

Simple radio stations widget

For this project you'll need Yarn (<https://classic.yarnpkg.com/en/docs/install/>).

## Technologies used

- `jest` for snapshot tests
- `eslint` and `prettier` for code formatting and validation
- `husky` was used to run commands pre-commit: eslint check and jest tests.
- `css-modules` for cleaner css
- `webpack` came with CRA, used for bundling files
- `typescript` for strict types

## Commands

### Running development

- `yarn`
- `yarn start`

And it should open in your default browser

### To run tests

- `yarn test`

### To build

- `yarn build`
  For deployment check <https://cra.link/deployment>

Git commit and push works as usual.
